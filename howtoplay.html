Instructions

  • Firstly, the game shows the first colour in the sequence (lets say blue), then you have to click on the blue button.
  • Next, the game shows the next colour (lets say red), you have to remember the sequence is blue, red and press the buttons accordingly.
  • Then the game chows next color (leys say green), you have to remember the sequence from previous levels and press the buttons in order to to advance in the game.
  • Repeat the process so on and so forth.
  • If you mess up the order, the game ends.
